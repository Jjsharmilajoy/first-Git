## Setup
Install dependencies
```sh
$ npm install
```

## Development
Connect to a virtual device or development device.
1. cd instabike-app/
2. react-native start --reset-cache
3. react-native run-android

## Deployment
1. cd instabike-app/android/
2. ./gradlew clean
3. ./gradlew assembleRelease
